# Assets Directory

This folder contains shared visual assets and front-end design references for EverNet VMM.

## Structure

- `assets/brand/`: logos and brand marks
- `assets/icons/`: functional UI icons
- `assets/tokens/`: design tokens (colors, spacing, radius, shadow)
- `assets/ui/`: reusable decorative patterns
- `assets/mock/`: non-sensitive sample UI payloads

## Usage Notes

1. Keep all files in SVG, CSS, or JSON where possible.
2. Do not store provider secrets or production credentials in this folder.
3. Name files with lowercase and kebab-case.
4. Reuse design tokens before introducing new hard-coded colors.
